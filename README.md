# Supply Chain Analytics

## Overview
This project analyzes the performance of a fashion and beauty supply chain using SQL and Tableau.

## Tools
- MySQL
- Tableau

## Dataset
Supply Chain Dataset (Kaggle) - https://www.kaggle.com/datasets/harshsingh2209/supply-chain-analysis

## Business Questions
- Which SKUs generate the highest revenue?
- Which suppliers have the highest defect rates?
- Which product types drive the most sales?
- Which carriers have the longest shipping times?
- Which high-revenue SKUs have inventory shortages?
